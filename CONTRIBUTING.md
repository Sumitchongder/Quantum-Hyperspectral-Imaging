# Contributing to Q-HSI

Thank you for your interest in the **Quantum-Enhanced Hyperspectral Imaging** project! We welcome contributions from the Quantum and Medical AI communities.

## How Can I Contribute?

### Reporting Bugs
* Use the **GitHub Issues** tab to report bugs.
* Include a clear title, a description of the bug, and steps to reproduce the error.
* State your OS, Python version, and key library versions (PyTorch, PennyLane).

### Suggesting Enhancements
* We are specifically looking for improvements in:
    * Quantum ansatz expressibility.
    * Multi-class classification (expanding beyond binary).
    * Integration with other QML frameworks like Qiskit Machine Learning.

### Pull Requests (PRs)
1. **Fork** the repository and create your branch from `main`.
2. Ensure your code follows **PEP 8** style guidelines.
3. If you've added code that should be tested, add tests.
4. Ensure the documentation (README) is updated if you change the architecture.
5. Open the PR with a clear description of the changes.

## Development Setup
1. Fork the repo.
2. Install dev dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the existing Jupyter Notebook to ensure the baseline environment is functional.
